Privacidad

Politica de privacidad

PDF Cambio procesa los archivos que el usuario sube unicamente para realizar la conversion solicitada. En esta version no se crea una cuenta de usuario ni se guarda historial publico de conversiones.

Archivos subidos

Los PDFs se reciben en el servidor para convertirlos a DOCX, JPG o PNG. La aplicacion esta pensada para devolver el archivo convertido en la misma solicitud y no conservar copias permanentes.

Publicidad y analitica

Mas adelante se podran integrar servicios de publicidad o analitica, como Google AdSense o Google Analytics. Si se agregan, esta politica debe actualizarse para explicar el uso de cookies y tecnologias similares.

Datos de pago

Cuando se active el plan Pro, los pagos deberan procesarse mediante una pasarela externa segura. PDF Cambio no debe almacenar numeros completos de tarjetas bancarias.

Contacto

Antes de publicar, reemplaza este texto con el correo real de contacto del proyecto.
