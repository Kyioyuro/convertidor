Uso del servicio

Terminos de uso

Al usar PDF Cambio, el usuario confirma que tiene derecho a subir y convertir los archivos que procesa en la plataforma.

Conversiones

La version gratuita permite 2 conversiones diarias. El plan Pro esta pensado para conversiones ilimitadas, sujeto a uso razonable, capacidad tecnica del servidor y politicas de seguridad.

Resultados

La conversion a Word prioriza conservar el diseno original insertando cada pagina como imagen dentro del documento. Para texto completamente editable puede requerirse una version futura con OCR y reconstruccion avanzada.

Uso prohibido

No se permite usar la herramienta para procesar archivos ilegales, maliciosos, sin autorizacion o que vulneren derechos de terceros.

Cambios

Estos terminos pueden actualizarse antes de la publicacion final, especialmente cuando se integren pagos, cuentas de usuario o publicidad.
